# domain_references

This directory contains background mathematical and project reference material.

These files are not operational prompts.
They are supporting references for problem design, evaluation, and mathematical review.

- `bessel_functions_guide.md`

- `spherical_harmonics_guide.md`

- `Probability_Crash_Course_RLHF.md`

- `linear_algebra_reference.md`

- `functional_analysis_reference.md`

## Usage

Use these files only as background support when needed.

Do not treat them as the active prompt system.

The active operational prompts are in:
- `domain_guides/`

The legacy prompt files are in:
- `domain_guides_legacy/`
