Let $L_n^{(\alpha)}(z)$ be the Laguerre polynomials. For $\text{Re}(\alpha) > -1$ and $z \in \mathbb{C} \setminus [0, \infty)$, the Laguerre function of the second kind is defined by:

$$Q_n^{(\alpha)}(z) = \int_0^\infty \frac{t^\alpha e^{-t} L_n^{(\alpha)}(t)}{z - t} dt$$


Define the function:

$$\phi_n(z) = z \frac{d}{dz} Q_n^{(\alpha)}(z) + (n+1) Q_{n+1}^{(\alpha)}(z) - (n + \alpha + 1 - z) Q_n^{(\alpha)}(z)$$

Claim: For all $n \ge 0$, the function $\phi_n(z)$ is identically zero on its domain.

Determine whether the claim is true or false, and give a rigorous proof.

conclusion: false

fix re$(\alpha)>-1$ and $z\in\mathbb{C}\setminus[0,\infty)$.

write

$$Q_n^{(\alpha)}(z)=\int_0^\infty\frac{w(t)L_n^{(\alpha)}(t)}{z-t} dt, \qquad w(t)=t^\alpha e^{-t}.$$

step 1. differentiate under the integral

since $z\notin[0,\infty)$, the integrand is analytic in $z$ and the integral converges absolutely, hence

$$Q_n^{(\alpha)\prime}(z) = -\int_0^\infty\frac{w(t)L_n^{(\alpha)}(t)}{(z-t)^2} dt.$$

using

$$\frac{z}{(z-t)^2}=\frac{1}{z-t}+\frac{t}{(z-t)^2},$$

we obtain

$$zQ_n^{(\alpha)\prime}(z) = Q_n^{(\alpha)}(z) + \int_0^\infty\frac{w(t)tL_n^{(\alpha)}(t)}{(z-t)^2} dt.$$

step 2. integration by parts

let

$$I:=\int_0^\infty\frac{w(t)tL_n^{(\alpha)}(t)}{(z-t)^2} dt.$$

since

$$\frac{d}{dt}\frac{1}{z-t}=\frac{1}{(z-t)^2},$$

we write

$$I=\int_0^\infty w(t)tL_n^{(\alpha)}(t)\frac{d}{dt}\left(\frac{1}{z-t}\right)dt.$$

integration by parts gives

$$I=\left[w(t)tL_n^{(\alpha)}(t)\frac{1}{z-t}\right]_0^\infty - \int_0^\infty\frac{(w(t)tL_n^{(\alpha)}(t))'}{z-t} dt.$$

because re$(\alpha)>-1$, one has $t^{\alpha+1}e^{-t}L_n^{(\alpha)}(t)\to0$ as $t\to0^+$ and as $t\to\infty$, so the boundary term vanishes. using the standard laguerre identity

$$(t^{\alpha+1}e^{-t}L_n^{(\alpha)}(t))' = (n+1)t^\alpha e^{-t}L_{n+1}^{(\alpha)}(t),$$

equivalently

$$(w(t)tL_n^{(\alpha)}(t))' = (n+1)w(t)L_{n+1}^{(\alpha)}(t),$$

we obtain

$$I = -(n+1)\int_0^\infty\frac{w(t)L_{n+1}^{(\alpha)}(t)}{z-t} dt = -(n+1)Q_{n+1}^{(\alpha)}(z).$$

step 3. combine

substituting $I$ into step 1 yields

$$zQ_n^{(\alpha)\prime}(z) = Q_n^{(\alpha)}(z)-(n+1)Q_{n+1}^{(\alpha)}(z),$$

hence

$$zQ_n^{(\alpha)\prime}(z) + (n+1)Q_{n+1}^{(\alpha)}(z) - Q_n^{(\alpha)}(z)=0.$$

step 4. evaluate $\phi_n$

by definition,

$$\phi_n(z) = zQ_n^{(\alpha)\prime}(z) + (n+1)Q_{n+1}^{(\alpha)}(z) - (n+\alpha+1-z)Q_n^{(\alpha)}(z).$$

using step 3, the first two terms equal $Q_n^{(\alpha)}(z)$, so

$$\phi_n(z) = Q_n^{(\alpha)}(z) - (n+\alpha+1-z)Q_n^{(\alpha)}(z) = (z-n-\alpha)Q_n^{(\alpha)}(z).$$

since $Q_n^{(\alpha)}$ is not identically zero (for example, for any real $z<0$ the defining integral is a nonzero real number), $\phi_n$ cannot vanish identically on $\mathbb{C}\setminus[0,\infty)$.

final answer: false
