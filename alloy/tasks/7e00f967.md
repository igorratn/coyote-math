## Prompt
Prove the following:

Let $\leq$ be a monomial order on $S$. Then the following holds:
(i) if $u, v \in \operatorname{Mon}(S)$ with $u \mid v$, then $u \leq v$;
(ii) if $u_{1}, u_{2}, \ldots$ is a sequence of monomials with $u_{1} \geq u_{2} \geq \cdots$, then there exists an integer $m$ such that $u_{i}=u_{m}$ for all $i \geq m$.

## Response 1
$S = k[x_1, \ldots, x_n]$, $\operatorname{Mon}(S) = \{x^{\alpha} := x_1^{\alpha_1} \cdots x_n^{\alpha_n} ; \alpha = (\alpha_1, \ldots, \alpha_n) \in \mathbb{N}^n\}$.

A monomial order $\leq$ on $\operatorname{Mon}(S)$ is a total order such that
$$(M1)\quad 1 \leq u \quad\text{for every } u \in \operatorname{Mon}(S),$$
$$(M2)\quad u \leq v \implies uw \leq vw \quad\text{for all } u,v,w \in \operatorname{Mon}(S).$$

### (i) Divisibility.
If $u \mid v$ there is a monomial $w$ with $v = uw$. By (M1) $1 \leq w$; multiplying this inequality by the monomial $u$ and using (M2) we obtain
$$u = u \cdot 1 \leq u \cdot w = v.$$
Hence $u \mid v$ implies $u \leq v$.

### (ii) No infinite strictly decreasing sequences.
Assume, to the contrary, that there is a strictly decreasing infinite sequence
$$(1)\quad u_1 > u_2 > u_3 > \cdots, \quad u_i = x^{\alpha^{(i)}}, \; \alpha^{(i)} = (\alpha_1^{(i)}, \ldots, \alpha_n^{(i)}) \in \mathbb{N}^n.$$
If we find indices $i < j$ such that $\alpha_k^{(i)} \leq \alpha_k^{(j)}$ for every $k$ then $u_i \mid u_j$; by (i) this gives $u_i \leq u_j$, contradicting (1). Hence it suffices to prove the following combinatorial lemma.

"Dickson's Lemma.
Every infinite sequence $(\beta^{(i)})_{i \geq 1} \subset \mathbb{N}^n$ contains indices $p < q$ with $\beta^{(p)} \leq \beta^{(q)}$ coordinate-wise."

Proof by induction on $n$.

For $n = 1$ the set $\mathbb{N}$ is well ordered; an infinite sequence of natural numbers cannot be strictly decreasing, so some $p < q$ satisfies $\beta^{(p)} \leq \beta^{(q)}$.

Suppose the statement true for $n - 1$ ($n \geq 2$) and let $\beta^{(i)} = (\beta_1^{(i)}, \ldots, \beta_n^{(i)})$ be an infinite sequence in $\mathbb{N}^n$. Consider the first coordinates $\beta_1^{(i)}$. Either the set $\{i : \beta_1^{(i)} \geq \beta_1^{(1)}\}$ is infinite; then choose the smallest $i_2 > 1$ with that property, and inductively obtain an infinite subsequence $i_1 = 1 < i_2 < i_3 < \cdots$ such that
$$(2)\quad \beta_1^{(i_1)} \leq \beta_1^{(i_2)} \leq \beta_1^{(i_3)} \leq \cdots.$$
If the above set is finite, then all but finitely many terms of the original sequence satisfy $\beta_1^{(i)} < \beta_1^{(1)}$; because the interval $\{0, \ldots, \beta_1^{(1)} - 1\}$ is finite, one integer occurs infinitely often, and we again obtain an infinite subsequence satisfying (2).

Relabel the subsequence as $(\gamma^{(m)})$. By construction its first coordinates are non-decreasing. Apply the induction hypothesis to the $(n-1)$-tuples $(\gamma_2^{(m)}, \ldots, \gamma_n^{(m)})$; we obtain $p < q$ with $\gamma_k^{(p)} \leq \gamma_k^{(q)}$ for $k = 2, \ldots, n$. Together with (2) for $k = 1$ we have $\gamma^{(p)} \leq \gamma^{(q)}$ componentwise, and hence $\beta^{(i_p)} \leq \beta^{(i_q)}$. ∎

Returning to (1), the lemma applied to the exponent vectors $\alpha^{(i)}$ yields $i < j$ with $u_i \mid u_j$. By part (i) this gives $u_i \leq u_j$, contradicting the strict decrease $u_i > u_j$. Therefore a strictly decreasing infinite sequence of monomials cannot exist.

Now let $u_1 \geq u_2 \geq \cdots$ be any decreasing sequence. If it were not eventually constant we could delete repetitions and obtain a strictly decreasing infinite subsequence, which we have just shown is impossible. Hence there is an $m$ such that $u_i = u_m$ for all $i \geq m$.

Thus a monomial order has no infinite strictly decreasing chain; it is a well-order on $\operatorname{Mon}(S)$. ∎

## Response 2
$S = K[x_1, \ldots, x_n]$, $\operatorname{Mon}(S) = \{x^{\alpha} = x_1^{a_1} \cdots x_n^{a_n} ; \alpha = (a_1, \ldots, a_n) \in \mathbb{N}^n\}$.

A monomial order is a total order $\leq$ on $\operatorname{Mon}(S)$ such that
$$(1)\quad 1 \leq x^{\alpha} \quad\text{for every } \alpha \in \mathbb{N}^n$$
and
$$(2)\quad x^{\alpha} \leq x^{\beta} \implies x^{\alpha}x^{\gamma} \leq x^{\beta}x^{\gamma} \quad \forall \alpha, \beta, \gamma \in \mathbb{N}^n.$$
Write $\alpha \leq_m \beta$ for $x^{\alpha} \leq x^{\beta}$; then (2) is $\alpha \leq_m \beta \Rightarrow \alpha + \gamma \leq_m \beta + \gamma$.

### (i)
Let $u, v \in \operatorname{Mon}(S)$ and $u \mid v$. Write $u = x^{\alpha}$, $v = x^{\beta}$. Since $u \mid v$ there is $\gamma \in \mathbb{N}^n$ with $\beta = \alpha + \gamma$, i.e. $v = uw$ with $w = x^{\gamma}$. From (1) we have $0 \leq_m \gamma$; adding $\alpha$ and using (2),
$$\alpha = \alpha + 0 \leq_m \alpha + \gamma = \beta.$$
Hence $u \leq v$. ∎

### (ii)
Assume that a strictly decreasing infinite sequence existed,
$$(3)\quad u_1 > u_2 > u_3 > \cdots, \quad u_i = x^{\alpha_i}, \; \alpha_i \in \mathbb{N}^n.$$
We use Dickson's Lemma.

"Lemma. Every subset of $\mathbb{N}^n$ has only finitely many minimal elements for the product order $\alpha \leq_p \beta \iff a_j \leq b_j \;(1 \leq j \leq n)$. Consequently any infinite sequence $(\alpha_i)$ in $\mathbb{N}^n$ contains indices $i < j$ with $\alpha_i \leq_p \alpha_j$."

Proof of the lemma. For $n = 1$ the claim is obvious. Let $n > 1$ and write $\alpha = (a, \beta)$ with $a \in \mathbb{N}$, $\beta \in \mathbb{N}^{n-1}$. If $(\alpha_i)$ is infinite, the numbers $a_i$ contain a non-decreasing subsequence; restricting to that subsequence and applying the induction hypothesis to the $\beta_i$'s gives the required pair. ∎

Apply the lemma to the exponent vectors $\alpha_i$ of (3). There are $i < j$ with $\alpha_i \leq_p \alpha_j$; i.e. $a_k^{(i)} \leq a_k^{(j)}$ for every $k$. Hence $\alpha_j = \alpha_i + \gamma$ for some $\gamma \in \mathbb{N}^n$ and
$$u_j = x^{\alpha_j} = x^{\alpha_i} x^{\gamma} = u_i w, \quad w := x^{\gamma} \in \operatorname{Mon}(S),$$
so $u_i \mid u_j$. By part (i) we obtain $u_i \leq u_j$, contradicting the strict inequality $u_i > u_j$ from (3). Hence no infinite strictly decreasing sequence of monomials exists.

If $u_1 \geq u_2 \geq \cdots$ therefore the set $\{i \mid u_i > u_{i+1}\}$ is finite; let $m$ be the largest index with $u_m > u_{m+1}$ (or $m = 1$ if no strict inequality occurs). Then $u_m = u_{m+1} = u_{m+2} = \cdots$.

Thus every weakly decreasing sequence of monomials stabilises. ∎
