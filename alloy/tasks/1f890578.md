## Setup
**Lp-norms:** For a vector $v \in \mathbb{R}^d$, the $p$-norm is defined as $\norm{v}_p = \left(\sum_{i=1}^d |v_i|^p\right)^{1/p}$.

**Sum-of-Squares (SoS) Polynomials:** A polynomial $p(v)$ is a sum of squares (SoS) if there exist polynomials $q_1(v), \dots, q_m(v)$ such that $p(v) = \sum_{i=1}^m q_i(v)^2$. If a polynomial is a sum of squares, it is non-negative everywhere.

**SoS proofs:** The notation $\{Q(v)\} \vdash \{P(v)\}$ for polynomials $P, Q$ signifies that $P(v)$ can be proven to be non-negative from the assumption that $Q(v)$ is non-negative, using the sum-of-squares proof system. In the context of this paper, an inequality like $A \le B$ is said to have an SoS proof if the polynomial $B-A$ is a sum of squares. The notation $\sststile{v} \Set{A \le B}$ is used to denote that $B-A$ is a sum of squares.

**Pseudo-distributions and Pseudo-expectations:** A degree-$d$ pseudo-distribution $\mu$ is a linear functional on the space of polynomials of degree at most $d$. The pseudo-expectation operator $\pexpecf{\mu}{\cdot}$ satisfies:
1.  Linearity: $\pexpecf{\mu}{a P + b Q} = a \pexpecf{\mu}{P} + b \pexpecf{\mu}{Q}$.
2.  Normalization: $\pexpecf{\mu}{1} = 1$.
3.  Positivity: $\pexpecf{\mu}{P^2} \ge 0$ for any polynomial $P$ such that $\deg(P^2) \le d$. This implies that for any SoS polynomial $S$, $\pexpecf{\mu}{S} \ge 0$.
4.  Pseudo-Jensen's Inequality: For any polynomial $P$ and integer $p \ge 1$ such that $\deg(P^p)$ is within the degree limit of the pseudo-distribution, $(\pexpecf{\mu}{P})^p \le \pexpecf{\mu}{P^p}$.
