### A1. Curvature of Space Curves in $\mathbb{R}^3$

**Total files: 3**

Computing curvature $\kappa(s)$ and torsion $\tau(s)$ using Frenet-Serret formulas.

#### A1.1 Direct Computation from Arc Length Parameterization

**Total files: 3**

**Typical Example: [b44b0a25.md](https://github.com/igorratn/coyote-math/blob/main/b44b0a25.md)**

Compute curvature $\kappa(s)$ and torsion $\tau(s)$ for the helix

$$\gamma(s) = \left(a\cos\frac{s}{c}, a\sin\frac{s}{c}, \frac{bs}{c}\right)$$

where $s$ is arc length and $a^2 + b^2 = c^2$.

**Solution Methodology:** Uses Frenet-Serret formulas relating derivatives of the moving frame $(T, N, B)$ to curvature and torsion. First verify $s$ is arc length: $\|\gamma'(s)\| = \sqrt{(a/c)^2\sin^2(s/c) + (a/c)^2\cos^2(s/c) + (b/c)^2} = \sqrt{(a^2+b^2)/c^2} = 1$. Tangent vector: $T(s) = \gamma'(s) = (-\frac{a}{c}\sin\frac{s}{c}, \frac{a}{c}\cos\frac{s}{c}, \frac{b}{c})$. Curvature is $\kappa(s) = \|T'(s)\| = \|\gamma''(s)\|$. Computing $\gamma''(s) = (-\frac{a}{c^2}\cos\frac{s}{c}, -\frac{a}{c^2}\sin\frac{s}{c}, 0)$ gives $\kappa = a/c^2$. Normal: $N(s) = T'(s)/\kappa = (-\cos\frac{s}{c}, -\sin\frac{s}{c}, 0)$. Binormal: $B(s) = T \times N$. Torsion from $\tau(s) = -N'(s) \cdot B(s)$ yields $\tau = b/c^2$.

**Answer:** $\kappa = a/c^2$, $\tau = b/c^2$

**Other files:**
- [08c9d3f8.md](https://github.com/igorratn/coyote-math/blob/main/08c9d3f8.md): Parabola $y = x^2$ curvature in $\mathbb{R}^2$ using $\kappa = |y''|/(1+y'^2)^{3/2}$. **(Answer: $\kappa(x) = 2/(1+4x^2)^{3/2}$)**
- [a8f9e1c7.md](https://github.com/igorratn/coyote-math/blob/main/a8f9e1c7.md): Twisted cubic $(t, t^2, t^3)$ torsion via Frenet formulas. **(Answer: $\tau(t) = 2/(1+4t^2+9t^4)$)**

### A2. Flat Tori & Quotient Manifolds

**Total files: 3**

Studying flat tori $T^2 = \mathbb{R}^2/\Lambda$ where $\Lambda$ is a lattice.

#### A2.1 Geodesic Length via Lattice Points

**Total files: 1**

**Typical Example: [c3e8d4f9.md](https://github.com/igorratn/coyote-math/blob/main/c3e8d4f9.md)**

Flat torus $T^2 = \mathbb{R}^2/\mathbb{Z}^2$ from unit square $[0,1] \times [0,1]$.

**Problem:** Compute length of shortest closed geodesic through $[(1/2, 1/2)]$ in direction $\theta = \pi/4$.

**Solution Methodology:** Lifts geodesic to universal cover $\mathbb{R}^2$. In universal cover, geodesics are straight lines (zero curvature). Line through $(1/2, 1/2)$ at angle $\pi/4$ has equation $y = x$, hitting lattice points $(n,n)$ for $n \in \mathbb{Z}$. Geodesic closes when lifted line returns to fundamental domain $[0,1) \times [0,1)$ modulo $\mathbb{Z}^2$. First return at $(1,1) \equiv (0,0)$. Distance in $\mathbb{R}^2$: $\|(1,1) - (0,0)\| = \sqrt{2}$.

**Answer:** $\sqrt{2}$

#### A2.2 Fundamental Domain Construction

**Total files: 1**

**Typical Example: [e5d3c4a2.md](https://github.com/igorratn/coyote-math/blob/main/e5d3c4a2.md)**

Lattice $\Lambda = \langle(1,0), (a,b)\rangle$ in $\mathbb{R}^2$ where $b \neq 0$.

**Problem:** Describe fundamental domain for $\mathbb{R}^2/\Lambda$.

**Solution Methodology:** Fundamental domain contains exactly one representative from each equivalence class. Since lattice generated by $v_1 = (1,0)$ and $v_2 = (a,b)$, fundamental domain is parallelogram with vertices $\{(0,0), (1,0), (a,b), (1+a,b)\}$. Any point written uniquely as $sv_1 + tv_2$ where $s,t \in [0,1)$ modulo boundary. Area equals $|\det(v_1, v_2)| = |1 \cdot b - 0 \cdot a| = |b|$.

**Answer:** Parallelogram with area $|b|$

#### A2.3 Counting Closed Geodesics

**Total files: 1**

**Typical Example: [07b9c3e1.md](https://github.com/igorratn/coyote-math/blob/main/07b9c3e1.md)**

Rectangular torus $T^2 = \mathbb{R}^2/\langle(L_1,0), (0,L_2)\rangle$.

**Problem:** Count distinct closed geodesics of length $\leq L$.

**Solution Methodology:** Closed geodesics correspond to straight lines connecting origin to lattice points $(mL_1, nL_2)$. Length is $\sqrt{(mL_1)^2 + (nL_2)^2}$. Distinct geodesics have different slopes (different ratios $n/m$ in lowest terms). Count primitive lattice points $\gcd(m,n)=1$ satisfying $\sqrt{(mL_1)^2 + (nL_2)^2} \leq L$. Asymptotic count uses density of primitive points: $N(L) \sim \frac{6L^2}{\pi L_1 L_2}$ as $L \to \infty$.

**Answer:** $N(L) \sim \frac{6L^2}{\pi L_1 L_2}$

### A3. Hyperbolic Geometry

**Total files: 3**

Hyperbolic plane $\mathbb{H}^2$ with constant negative curvature $K = -1$.

#### A3.1 Gauss-Bonnet Formula Applications

**Total files: 2**

**Typical Example: [bce7d3f2.md](https://github.com/igorratn/coyote-math/blob/main/bce7d3f2.md)**

Geodesic triangle $\Delta$ in $\mathbb{H}^2$ with interior angles $\alpha, \beta, \gamma$.

**Problem:** Compute area of $\Delta$.

**Solution Methodology:** Applies Gauss-Bonnet theorem for constant curvature. For geodesic polygon with $n$ vertices and interior angles $\alpha_1, \ldots, \alpha_n$ on surface with Gaussian curvature $K$:

$$\int_{\text{polygon}} K \, dA + \sum_{i=1}^n (\pi - \alpha_i) = (n-2)\pi$$

For hyperbolic geometry $K = -1$, so $\int K \, dA = -\text{Area}$. For triangle ($n=3$):

$$-\text{Area} + (\pi - \alpha) + (\pi - \beta) + (\pi - \gamma) = \pi$$

Solving: $\text{Area} = \pi - (\alpha + \beta + \gamma)$. The **angle deficit** $\pi - (\alpha+\beta+\gamma)$ is positive since hyperbolic angle sum $< \pi$.

**Answer:** $\text{Area} = \pi - (\alpha + \beta + \gamma)$

**Other files:**
- [c4e9f8a3.md](https://github.com/igorratn/coyote-math/blob/main/c4e9f8a3.md): Hyperbolic disk radius $R$; integrates metric to get $\text{Area} = 2\pi(\cosh R - 1)$. **(Answer: $2\pi(\cosh R - 1)$)**

#### A3.2 Distance Formulas

**Total files: 1**

**Typical Example: [d5f3e8c2.md](https://github.com/igorratn/coyote-math/blob/main/d5f3e8c2.md)**

Upper half-plane model $\mathbb{H}^2 = \{z \in \mathbb{C} : \text{Im}(z) > 0\}$ with metric $ds^2 = \frac{dx^2 + dy^2}{y^2}$.

**Problem:** Compute hyperbolic distance between $z_1 = i$ and $z_2 = 2i$.

**Solution Methodology:** For points $z_1 = iy_1$, $z_2 = iy_2$ on imaginary axis with $y_2 > y_1 > 0$, geodesic is vertical line segment (orthogonal to real axis). Distance by integrating metric:

$$d(z_1, z_2) = \int_{y_1}^{y_2} \frac{dy}{y} = \ln\left(\frac{y_2}{y_1}\right)$$

For $z_1 = i$ ($y_1=1$) and $z_2 = 2i$ ($y_2=2$): $d = \ln(2/1) = \ln 2$. Logarithmic formula reflects exponential growth: fixed hyperbolic distance requires exponential Euclidean displacement in $y$.

**Answer:** $\ln 2$

### B1. Modified Helmholtz Equation

**Total files: 2**

Equation $\Delta u - k^2 u = f$ with exponential decay.

#### B1.1 Fourier Transform Methods

**Total files: 2**

**Typical Example: [f8e3c7b5.md](https://github.com/igorratn/coyote-math/blob/main/f8e3c7b5.md)**

Solve $u_{xx} + u_{yy} - k^2 u = f(x,y)$ on half-plane $y > 0$ with $u(x,0) = g(x)$ and $u \to 0$ as $y \to \infty$.

**Solution Methodology:** Uses Fourier transform in $x$: $\hat{u}(\xi, y) = \int u(x,y) e^{-i\xi x} dx$. PDE becomes ODE: $\hat{u}_{yy} - (\xi^2 + k^2)\hat{u} = \hat{f}$. Homogeneous solution: $\hat{u}_h = Ae^{\sqrt{\xi^2+k^2}y} + Be^{-\sqrt{\xi^2+k^2}y}$ where $\sqrt{\xi^2+k^2} > 0$ for all real $\xi$. Decay condition $u \to 0$ as $y \to \infty$ requires $A=0$ (growing exponential vanishes). Boundary condition $u(x,0) = g(x)$ transforms to $\hat{u}(\xi,0) = \hat{g}(\xi)$, giving $B = \hat{g}(\xi)$. Thus:

$$\hat{u}(\xi, y) = \hat{g}(\xi) e^{-\sqrt{\xi^2+k^2}y}$$

Inverse Fourier transform:

$$u(x,y) = \frac{1}{2\pi}\int_{-\infty}^\infty \hat{g}(\xi) e^{-\sqrt{\xi^2+k^2}y} e^{i\xi x} d\xi$$

For $f \neq 0$, add particular solution via variation of parameters.

**Answer:** Explicit integral formula in terms of $\hat{g}(\xi)$

**Other files:**
- [e7c5f8a3.md](https://github.com/igorratn/coyote-math/blob/main/e7c5f8a3.md): Strip domain $0 < x < L$ with $u(0,y) = u(L,y) = 0$; uses Fourier sine series $u = \sum u_n(y)\sin(n\pi x/L)$ where each $u_n$ satisfies ODE. **(Answer: series solution)**

### B2. Laplace's Equation

**Total files: 1**

#### B2.1 Separation of Variables in Rectangles

**Total files: 1**

**Typical Example: [c8b5e7f3.md](https://github.com/igorratn/coyote-math/blob/main/c8b5e7f3.md)**

Solve $\Delta u = 0$ in rectangle $[0,a] \times [0,b]$ with Dirichlet conditions: $u(0,y)=u(a,y)=0$, $u(x,0)=f(x)$, $u(x,b)=g(x)$.

**Solution Methodology:** Separation of variables with $u = X(x)Y(y)$ gives $\frac{X''}{X} = -\frac{Y''}{Y} = \lambda$. Boundary conditions $X(0)=X(a)=0$ yield eigenvalues $\lambda_n = (n\pi/a)^2$ with eigenfunctions $X_n(x) = \sin(n\pi x/a)$ for $n = 1,2,3,\ldots$ For each $\lambda_n$, solve $Y'' - (n\pi/a)^2 Y = 0$: $Y_n(y) = A_n \sinh(n\pi y/a) + B_n \cosh(n\pi y/a)$. General solution:

$$u(x,y) = \sum_{n=1}^\infty [A_n \sinh(n\pi y/a) + B_n \cosh(n\pi y/a)] \sin(n\pi x/a)$$

Boundary conditions at $y=0$ and $y=b$ determine coefficients via Fourier sine series:

$$B_n = \frac{2}{a}\int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) dx$$

$$A_n = \frac{1}{\sinh(n\pi b/a)}\left[\frac{2}{a}\int_0^a g(x) \sin\left(\frac{n\pi x}{a}\right) dx - B_n \cosh\left(\frac{n\pi b}{a}\right)\right]$$

**Answer:** Fourier series solution with explicit coefficients

#### C1.1 Frobenius Series at Regular Singular Points

**Total files: ~3**

**Typical Example: [Placeholder file to be identified]**

Standard hypergeometric equation:

$$z(1-z)w'' + [c-(a+b+1)z]w' - abw = 0$$

with solutions ${}_2F_1(a,b;c;z)$.

**Solution Methodology:** Uses Frobenius series expansion at regular singular points $z=0, 1, \infty$. At $z=0$, substitute $w = z^\rho \sum_{n=0}^\infty a_n z^n$. Indicial equation $\rho(\rho-1+c) = 0$ gives exponents $\rho_1 = 0$ and $\rho_2 = 1-c$. For $c \notin \mathbb{Z}$, two independent solutions:

$$w_1(z) = {}_2F_1(a,b;c;z) = \sum_{n=0}^\infty \frac{(a)_n(b)_n}{(c)_n n!}z^n$$

$$w_2(z) = z^{1-c}{}_2F_1(a-c+1, b-c+1; 2-c; z)$$

where $(a)_n = a(a+1)\cdots(a+n-1)$ is Pochhammer symbol. Recurrence relation for coefficients: $a_{n+1}/a_n = \frac{(a+n)(b+n)}{(c+n)(n+1)}$. Connection formulas relate solutions at different singular points. Integral representations (Euler, Barnes) provide alternative characterizations.

**Answer:** Solution in terms of ${}_2F_1$ hypergeometric function
