## File Inventory by Cluster

### orthogonal_polynomials.md — 43 files
Jacobi polynomial orthogonality and norms, Rodrigues formulas, Christoffel-Darboux identity, asymptotics (Mehler-Heine, large-$n$), Laguerre polynomial integrals and second kind, Hermite generating functions and oscillator wavefunctions, monotonicity of polynomial maxima.

### spherical_harmonics.md — 20 files
Legendre polynomial identities, addition theorem proofs, generating function series, associated Legendre recurrence, completeness of harmonic expansions.

### sine_gordon.md — 12 files
Damped sine-Gordon equilibria (Josephson junctions, DNA, ferromagnets), undamped/kink boundary value problems, Helmholtz-type reductions, static phase plane analysis.

### angular_momentum.md — 15 files
Clebsch-Gordan coefficients and Wigner 3j symbols, 6j symbols (Racah coefficients), triangle inequalities, orthogonality, Racah and Hahn polynomial representations, semiclassical/asymptotic limits.

### bessel_functions.md — 12 files
Bessel equation variants (Lommel, modified), Wronskian relations, orthogonality on intervals, asymptotics (Mehler-Heine, Hankel), branch cut analysis, spherical Bessel polynomial representation.

### differential_geometry.md — 9 files
Hyperbolic geometry (Gauss-Bonnet, Poincaré disk distances, holonomy on constant-curvature surfaces), surface curves (Darboux frame, geodesic curvature/torsion, Weingarten map), flat tori (quotient geodesics, parallel transport, geodesic optimization).

**Figure files:** 8 problems have associated figures following the `{id}-i.*` naming convention (`.jpg`, `.pdf`, `.tex`).

### miscellaneous_topics.md — 5 files
Fourier transform reduction of modified Helmholtz PDE, asymptotic expansions via Taylor series, infimum of implicit-equation sets, Christoffel-Darboux kernel degree analysis.

## Organization Principles

**Clustering by solution methodology** — problems are grouped by *how* they are solved, not *what topic* they cover. Each cluster document uses a three-level hierarchy:

- `##` Cluster — broad methodological approach
- `###` Sub-cluster — specific problem type
- `####` Method — precise solving technique, always with a "Typical Example"

**"Typical Example" format:** Full problem statement + 4–6 sentence methodology description + conclusion.
