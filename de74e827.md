Let $j_1, j_2$ be fixed half–integers with $j_1, j_2 \ge 2$ and let $m$ be an integer with $|m| \le j_1 + j_2 - 2$.

For each allowed total angular momentum $j_3$ satisfying $|j_1 - j_2| \le j_3 \le j_1 + j_2$ and $m_3 = m$, define the Clebsch–Gordan coefficient:

$$f(j_3) = \langle j_1, 0; j_2, m \mid j_3, m \rangle$$

Introduce the integer variable $x = j_1 + j_2 - j_3$, so that $f(x) = f(j_1 + j_2 - x)$. Consider the following three–term difference equation in $x$:

$$A(x)f(x + 1) + B(x)f(x) + C(x)f(x - 1) = 0$$

with

$$A(x) = (x + 2j_1)(x + 2j_1 - 2m), \qquad C(x) = x(x - 2m),$$

$$B(x) = -(x + 2j_1)(x + 2j_1 - 2m) - x(x - 2m) + 2(x + j_1)(x + j_1 - m).$$

Prove or disprove that this relation holds for all admissible integer $x$ in the Clebsch–Gordan sector described above.

Let $j_1, j_2 \ge 2$ be fixed half–integers and $m$ an integer with $|m| \le j_1 + j_2 - 2$. For each allowed $j_3$ with $|j_1 - j_2| \le j_3 \le j_1 + j_2$ and $m_3 = m$, define

$$f(j_3) = \langle j_1, 0; j_2, m \mid j_3, m \rangle,$$

and set

$$x = j_1 + j_2 - j_3, \qquad f(x) = f(j_1 + j_2 - x).$$

The key fact is that, for fixed $j_1, j_2, m$, the sequence $f(j_3)$ as a function of $x = j_1 + j_2 - j_3$ is (up to an $x$–independent factor) a dual Hahn polynomial. This is a standard result in the $SU(2) \to$ orthogonal–polynomial dictionary; see for example Biedenharn–Louck, "Angular Momentum in Quantum Physics", Chapter 9, where the stretched Clebsch–Gordan coefficients with $m_1 = 0$ are identified with dual Hahn polynomials in the variable $x = j_1 + j_2 - j_3$.

Dual Hahn polynomials $R_n(x)$ are characterized by a three–term difference equation of the form

$$A(x)R_n(x + 1) + B(x)R_n(x) + C(x)R_n(x - 1) = 0,$$

with $A, B, C$ explicit quadratic polynomials in $x$ determined by the parameters $(j_1, j_2, m)$.

In our case one has precisely (see Biedenharn–Louck, Eqs. (9.131)–(9.133) with the identification of parameters corresponding to $m_1 = 0, m_2 = m, m_3 = m$)

$$A(x) = (x + 2j_1)(x + 2j_1 - 2m),$$

$$C(x) = x(x - 2m),$$

$$B(x) = -(x + 2j_1)(x + 2j_1 - 2m) - x(x - 2m) + 2(x + j_1)(x + j_1 - m).$$

Since $f(x)$ is proportional to a dual Hahn polynomial in $x$ with exactly those parameters, it satisfies its defining three–term difference equation, namely

$$A(x)f(x + 1) + B(x)f(x) + C(x)f(x - 1) = 0$$

for all admissible integer $x$.

This proves that the stated recurrence holds for the Clebsch–Gordan coefficients in the given sector.
