## QC Flag Justifications for Rubric

Use these 1-3 sentence justifications to dismiss each "Atomic criterion" QC flag.

---

**Criterion 1** — "Identifies the Schrödinger-like radial equation for the upper Dirac component under spin symmetry with ℓ(ℓ+1) = 2 (from κ = −2, ℓ = 1)."
> The mapping κ = −2 → ℓ = 1 → ℓ(ℓ+1) = 2 is a single logical chain, not three independent concepts. Splitting it would create trivially dependent sub-criteria since each step follows immediately from the previous one. This criterion tests one evaluable skill: correctly setting up the radial equation.

**Criterion 2** — "Applies the Pekeris-type centrifugal approximation and substitutes z = e^{−αr}."
> The centrifugal approximation is defined in terms of the variable z = e^{−αr}, so one cannot apply the approximation without performing the substitution. These are not independently evaluable steps — they are a single methodological choice. Splitting would create a criterion ("substitute z") that has no standalone meaning without the approximation.

**Criterion 3** — "Defines the energy-dependent dimensionless parameters ε, η, β, γ, and δ."
> These five parameters form one coherent parametrization of the transcendental energy equation. They are not independently meaningful — defining ε without η, or β without γ, has no standalone evaluable value. The single evaluable question is: did the response correctly parametrize the energy-dependent quantization condition?

**Criterion 4** — "States the bound-state quantization condition for n_r = 0."
> Stating the general quantization formula and evaluating it at n_r = 0 is one step, not two. The general formula without the n_r = 0 substitution is not independently useful for solving this specific problem. The criterion tests one skill: writing the correct quantization condition for the requested ground state.

**Criterion 5** — "Solves the transcendental equation numerically to obtain E ≈ 249 MeV."
> The method (numerical iteration) and the result (E ≈ 249 MeV) are inseparable for evaluation purposes. A response that iterates numerically but gets the wrong answer has a different failure mode than one that attempts a closed-form solution. Grading guidance already distinguishes these cases with explicit point values.

**Criterion 6** — "Computes the binding energy E_b = mc² − E ≈ 689 MeV."
> The formula E_b = mc² − E and the numerical result 689 MeV are tested together because the formula is trivial (one subtraction) — the real test is whether the correct E was used. A response with the right formula but wrong E will already lose points on Criterion 5. Splitting would create redundant evaluation.
